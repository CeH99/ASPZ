# Пошук числа у динамічному масиві

Проста та ефективна програма мовою C, яка генерує двовимірний масив (матрицю) з випадковими дробовими числами та виконує пошук усіх входжень заданого числа.

## Функціонал
- **Динамічна пам'ять:** Використання `malloc` для створення масивів будь-якого розміру.
- **Аргументи командного рядка:** Розмір масиву задається при запуску програми.
- **Точність:** Коректне порівняння чисел з плаваючою точкою (`float`) з урахуванням похибки.

## Компіляція та запуск

1. **Скомпілюйте програму:**
   ```bash
   gcc main.c -o array_search
Запустіть (наприклад, 5 рядків і 5 стовпців):

Bash
./array_search 5 5
